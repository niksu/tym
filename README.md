# TYM Datalog
This is a simple implementation of Datalog.

# Building
`make`

or for debug output: `CFLAGS=-DDEBUG make`

# Running
`./tym`

e.g., `./tym -i tests/4.test`

The `-v` flag activates verbose output.

# Tests
`make test_units`
`make test_regression`
